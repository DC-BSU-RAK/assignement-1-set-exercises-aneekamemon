A Quick Guide on HTML & CSS

HTML stands for HyperText Markup Language. It is a standard markup language for web page creation. It allows the creation and structure of sections, paragraphs, and links using HTML elements (the building blocks of a web page) such as tags and attributes.

Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

How They Work:

  1. !DOCTYPE - Defines the document type
  2. body - Defines the document's body
  3. section - Defines a section in a document
  4. br - Defines a single line break
  5. footer - Defines a footer for a document or section
  6. head - Contains metadata/information for the document
  7. img - Defines an image
  8. input - Defines an input control
  9. ol - Defines an ordered list
  10. video - Defines embedded video content

  1. !DOCTYPE - 
    <!DOCTYPE html>
    <head>
  2. body - 
    <body>
    <h1> title /<h1>
    </body>
  3. section - 
    <section>
    <h3> Ex /<h3>
    </section>
  4. br - 
    <title> CSS <br> Example /<title>
  5. footer - 
    <footer> ©Copyright Reserved </footer>
  6. head - 
    <head>
    <h1> Ex /<h1>
    </head>
  7. img - 
    <img src= "http://image/.png" alt = "image">
  8. input - 
    <input type= "text" name= "BSU" value= "tab">
  9. ol - 
    <ol>
    <li> this is an item /<li>
    </ol>
  10. video - 
    <video controls>
    <source src= "http://video/.mp4" type = "video/mp4">
    </video>

  1. color - Defines the color of the text
  2. font-size - Controls the size of the font
  3. padding - Sets the spacing of the content and the border of the element
  4. text-align - Works to align texts inside an element
  5. display - Defines how an element should be displayed

  1. color - p1 {
    color: green;
    }
  2. font-size - a1 {
    font-size: 98px;
    }
  3. padding - title {
    padding: 20px;
    }
  4. text-align - h3 {
    text-align: center;
    }
  5. display - section {
    display: flex;
    }
